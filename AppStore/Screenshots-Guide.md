### Frame Template (Optional)
- Device frame: iPhone 15 Pro in Natural Titanium
- Background: Gradient from #0D3B2E to #1A5F4A
- Text position: Bottom 15% with padding

### Text Styling
- **Headline**: SF Pro Display Bold, 48pt, #FFFFFF
- **Subheadline**: SF Pro Text Regular, 24pt, #E8E8E8
- **Shadow**: 0 2px 20px rgba(0,0,0,0.5)

## Screenshot Content Guidelines

### Do's ✅
- Use actual app UI, not mockups
- Show real content (no placeholder text)
- Highlight key features clearly
- Maintain dark green color theme
- Include diverse models/content
- Ensure text is readable

### Don'ts ❌
- No pricing information
- No time-sensitive content
- No competitor mentions
- No misleading functionality
- No excessive text
- No violence, profanity, or adult content

## App Store Connect Upload Requirements

### File Specifications
- **Format**: PNG or JPEG
- **Color Space**: sRGB or P3
- **File Size**: Max 50MB per screenshot
- **No Alpha Channel**: Flattened images only

### Upload Order
1. iPhone 15 Pro Max (6.7" Display)
2. iPhone 15 Pro (6.1" Display)  
3. iPad Pro 12.9" (3rd gen and later)
4. iPad Pro 11" (1st gen and later)

### Localization
Prepare screenshots for:
- English (US) - Primary
- English (UK)
- English (Australia)
- English (Canada)

## Screenshot Production Checklist

- [ ] Capture on actual devices (not simulators for final)
- [ ] Verify all text is localized
- [ ] Check contrast ratios meet WCAG AA
- [ ] Ensure no status bar shows carrier/debug info
- [ ] Remove any test/prototype data
- [ ] Verify image dimensions match requirements exactly
- [ ] Run through App Store Connect validation
- [ ] Test appearance in dark mode (if applicable)

## Preview Video (Optional but Recommended)

### Specifications
- **Format**: H.264 or ProRes 422
- **Resolution**: Match target device
- **Length**: 15-30 seconds
- **Frame Rate**: 30fps
- **File Size**: Max 500MB

### Content Flow
1. 0-5s: Opening the app, browsing feed
2. 5-15s: Visual search demonstration
3. 15-25s: Listing an item with AI
4. 25-30s: Viewing sustainability impact
